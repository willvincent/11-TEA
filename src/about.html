About 11-TEA

What can I say? It's a starter for 11ty, that incorporates some of the best modern tooling available:

  • Tailwind CSS 3 - with all the great things it has to offer, namely JIT, how could we not?
  • AlpineJS 3 - Most sites need some JS. Many use too much. Alpine gets it done with less.
  • Lazy Images - Lazy loaded, responsive images
